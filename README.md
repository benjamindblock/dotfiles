# dotfiles
dotfiles managed with rcm

## Installation
1. `xcode-select --install`
2. `git clone git@github.com:benjamindblock/dotfiles.git .dotfiles`
3. `$HOME/.dotfiles/setup`
4. `vi .vimrc`
5. `:PlugInstall`
